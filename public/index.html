üß† How the Greedy Algorithm Works:

The algorithm maintains a variable maxReach that tracks the farthest position we can reach. At each step, if our current position exceeds maxReach, we're stuck and return false. Otherwise, we update maxReach = max(maxReach, currentIndex + jumpValue) and continue. Time complexity: O(n), Space complexity: O(1).
